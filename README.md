# swilc.github.io
Style Sheet Hosting

Dark mode code
----
  For dark mode, use:
  `<link rel="stylesheet" type="text/css" href="https://swilc.github.io/styles/styles-dark.css"/>`
  
Light mode code
----
  For light/standard, use:
  `<link rel="stylesheet" type="text/css" href="https://swilc.github.io/styles/styles.css"/>`
